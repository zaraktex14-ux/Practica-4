# Practica-4 AFD y Expresiones Regulares
## 📌 Descripción

Este proyecto consiste en implementar

* Autómatas Finitos Deterministas (**AFD**)
* Expresiones Regulares (**ER**)

El sistema permite visualizar, convertir y validar estructuras formales de manera gráfica e interactiva.

---

## 🎯 Objetivos

* Implementar la conversión de **AFD → ER**
* Visualizar el proceso paso a paso

---

# 🚀 Funcionalidades

## 🔹 Gestión de AFD

* Creación y edición mediante interfaz gráfica
* Visualización de estados y transiciones
* Guardado y carga de archivos
* Verificación de AFD válido

---

## 🔹 Conversión AFD → ER

* Algoritmo de eliminación de estados
* Proceso interactivo paso a paso
* Visualización clara de cada transformación
* Generación automática de la ER final

---

## 🔹 Importación desde JFLAP

* Lectura de archivos `.jff`
* Interpretación de autómatas
* Validación de AFD
* Visualización automática

---

## 🔹 Casos prácticos con ER

### Validadores incluidos:

* 📧 Correos electrónicos
* 📱 Números telefónicos
* 🌐 URLs
* 📅 Fechas
* 🔐 Contraseñas

### Funcionalidades:

* Mostrar expresión regular
* Validación de entrada
* Resultado claro (válido / inválido)
